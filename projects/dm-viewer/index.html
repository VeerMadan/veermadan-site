VePlexity’s Insta DM Viewer

Built for memories. Designed for clarity.

📌 What is this?
A private, offline tool to view and explore your Instagram DMs with advanced features like search, highlights, and stats.

⚡ How to use:

  1. Download your Instagram data (messages)
  2. Locate messages_1.json
  3. Upload it inside this tool

📥 How to download Instagram messages:
Instagram → Settings → Accounts Center → Your Info → Download Your Info → Request JSON

🔒 Privacy:
✔ Runs locally in your browser
✔ No data uploaded
✔ No storage / no tracking

💻 Offline Use:
Download from GitHub and run without internet.
